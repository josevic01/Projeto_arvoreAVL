Projeto desenvolvido em linguagem C com o objetivo de implementar um dicionário de sinônimos utilizando uma Árvore AVL para armazenamento e busca eficiente das palavras.

O sistema realiza a leitura de um arquivo de entrada contendo palavras e seus respectivos sinônimos, armazenando os dados em uma estrutura AVL balanceada. Após a construção da árvore, é possível realizar consultas de palavras e retornar seus sinônimos, além do caminho percorrido durante a busca na árvore.

O projeto foi desenvolvido com foco em praticar conceitos de Estruturas de Dados, especialmente Árvores AVL, além de reforçar conhecimentos sobre manipulação de memória dinâmica, recursividade e organização de dados em C.
